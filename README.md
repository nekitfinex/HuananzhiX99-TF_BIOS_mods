# HuananzhiX99-TF_BIOS_mods

Здесь представлен модифицированный биосы на материнскую плату X99-TF (rev 1.1) с вшитой поддержой TPM-модуля и Secure Boot. Первая версия биос имеет встроенный анлок турбобуста, а также андервольтинг (-50mV/-50mV). 

Внимание: эта версия биоса не запусится на процессорах V4 серии! Версия поддерживает работу на ddr3 и ddr4 памяти! 

Вторая версия биос имеет поддержку процессоров V4 серии, не имеет анлок турбобуста и имеет поддержку TPM и Secure Boot.

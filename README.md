# HuananzhiX99-TF_BIOS_mods
Здесь представлен модифицированный биос на материнскую плату X99-TF (rev 1.1) с вшитой поддержой TPM-модуля и Secure Boot. Эта версия биос имеет встроенный анлок турбобуста, а также андервольтинг (-50mV/-50mV). Внимание: эта версия биоса не запусится на процессорах V4 серии. Версия поддерживает работу на ddr3 и ddr4 памяти.
